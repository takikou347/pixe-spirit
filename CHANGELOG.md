# Changelog

## 0.1.0 (2026-03-18)

Initial release.

### Features

- Seed-deterministic pixel spirit generation (32x32 grid)
- 8 body shapes: circle, oval (tall/wide), rounded square, teardrop, dome, tall dome, creature
- 5 eye styles: dot, small, medium, large, cyclops
- 4 mouth styles: line, smile, open, none
- 8 decoration types: halo, horns, antennae, hat, leaf, crown, book
- 5 ear types: pointed, round, long, cat
- 4 tail types: bushy, thin, curled
- 4 marking patterns: dots, stripe, patch, speckles
- Traits override: specify some traits, randomize the rest from seed
- 8 color palettes (4 colors x light/dark) + custom palette support
- 4 preset characters: egg, baby, child_diary, child_balanced
- 4 animation types: idle, happy, sad, sleep
- Dual ESM/CJS build with TypeScript declarations
- GitHub Pages gallery for browsing generated characters
